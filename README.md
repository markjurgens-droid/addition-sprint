# Addition Sprint

A simple Grade 2 addition flashcard app for classroom review.

Students type the answer, press Enter to check it, and press Enter again after a correct answer to move to the next question.

## Files

- `index.html` - the app page
- `styles.css` - the visual design
- `app.js` - the flashcard logic

## GitHub Pages

This app can be served directly from GitHub Pages because it uses plain HTML, CSS, and JavaScript.
